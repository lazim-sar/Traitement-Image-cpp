# 🖼️ Traitement d'image — Filtres & Transformations

Projet académique réalisé en L1 Maths-Informatique à l'Université Paris-Saclay (2026),
dans le cadre du cours de Programmation Impérative (C++).

## Description
Programme de traitement d'images permettant d'appliquer différentes
transformations sur des images personnelles.

### Fonctionnalités
- Conversion couleur vers noir & blanc
- Extraction de contours
- Application de filtres personnalisés

## Technologies
- Langage : C++
- Paradigme : Programmation impérative

## Compilation & Exécution
```bash
g++ -o traitement main.cpp
./traitement
```
